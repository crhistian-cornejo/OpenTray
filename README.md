# OpenTray

Menubar companion for [OpenCode](https://opencode.ai). Monitor and control your AI coding sessions from the system tray.

## Download

Get the latest version from [Releases](https://github.com/crhistian-cornejo/OpenTray/releases).

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `.dmg` (aarch64) |
| macOS (Intel) | `.dmg` (x64) |
| Windows | `.exe` / `.msi` |

## Features

- System tray app (menubar on macOS)
- Auto-discover OpenCode instances
- Manage sessions (create, archive, delete)
- Real-time chat with streaming
- Diff viewer
- TODO tracking
- Native notifications
- Auto-updates

## Development

npm install
